<h2>
  
  
  What I Built
</h2>

<p>I built <strong>LifeHub</strong>, a cross-platform productivity and wellbeing assistant created using the <strong>Uno Platform</strong>.<br><br>
It combines daily planning, habit tracking, mood logging, reminders, and quick-action widgets into a clean and responsive UI.<br><br>
The entire interface was refined using the <strong>Hot Design Agent</strong>, which helped speed up UI experimentation and real-time adjustments.</p>

<h2>
  
  
  Design Process
</h2>

<ul>
<li>Started with a base Uno template and structured the main layout with <strong>Grid</strong> and <strong>StackPanel</strong>.
</li>
<li>Used <strong>Hot Reload</strong> extensively to adjust padding, spacing, and color themes without rebuilding.
</li>
<li>Set up <strong>MVVM bindings</strong> for tasks, habits, and mood entries.
</li>
<li>Implemented animations using <strong>Implicit Animations</strong> to add smooth transitions.
</li>
<li>Ensured adaptive behavior using <strong>AdaptiveTriggers</strong> for mobile, tablet, and desktop screens.
</li>
<li>Final UI refinements were done with feedback from the Hot Design Agent.</li>
</ul>
