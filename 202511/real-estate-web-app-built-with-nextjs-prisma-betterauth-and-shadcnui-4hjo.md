<h1>
  
  
  PropPulse
</h1>

<p><strong>PropPulse</strong> is a modern and minimal real estate web application** built with Next.js 16, Prisma,BetterAuth, and ShadCN/UI.<br><br>
Users can browse properties, add listings, and manage their real estate posts through a clean and fast interface.</p>

<h2>
  
  
  ğŸ›  Getting Started
</h2>

<p>Follow these steps to run the project locally:</p>
<h3>
  
  
  1. Clone the repository
</h3>


<div class="highlight js-code-highlight">
<pre class="highlight shell"><code>git clone https://github.com/saidMounaim/prop-pulse.git
<span class="nb">cd </span>prop-pulse
</code></pre>

</div>

<h3>
  
  
  2. Install dependencies
</h3>


<div class="highlight js-code-highlight">
<pre class="highlight shell"><code>npm <span class="nb">install</span>
</code></pre>

</div>

<h3>
  
  
  3. Configure environment variables
</h3>

<p>Create a <code>.env</code> file in the root:<br>
</p>

<div class="highlight js-code-highlight">
<pre class="highlight plaintext"><code># Database
DATABASE_URL="postgresql://..."

# BetterAuth
BETTER_AUTH_BASE_URL="https://proppulse-next.netlify.app"
BETTER_AUTH_SECRET="your_betterauth_secret"

# ImageKit
IMAGEKIT_PUBLIC_KEY="your_public_key"
IMAGEKIT_PRIVATE_KEY="your_private_key"
IMAGEKIT_URL_ENDPOINT="https://ik.imagekit.io/your_id"
</code></pre>

</div>



<h3>
  
  
  4. Start the dev server
</h3>



<div class="highlight js-code-highlight">
<pre class="highlight shell"><code>npm run dev
</code></pre>

</div>
