<h3>
  
  
  Basic Script Structure
</h3>

<p>A basic Lua script in Roblox starts with defining functions and variables. Here’s a simple example:<br>
</p>

<div class="highlight js-code-highlight">
<pre class="highlight lua"><code><span class="kd">local</span> <span class="k">function</span> <span class="nf">welcomePlayer</span><span class="p">(</span><span class="n">player</span><span class="p">)</span>
    <span class="nb">print</span><span class="p">(</span><span class="s2">"Welcome to the game, "</span> <span class="o">..</span> <span class="n">player</span><span class="p">.</span><span class="n">Name</span> <span class="o">..</span> <span class="s2">"!"</span><span class="p">)</span>
<span class="k">end</span>

<span class="n">game</span><span class="p">.</span><span class="n">Players</span><span class="p">.</span><span class="n">PlayerAdded</span><span class="p">:</span><span class="n">Connect</span><span class="p">(</span><span class="n">welcomePlayer</span><span class="p">)</span>
</code></pre>

</div>



<p>This script welcomes players as they join the game. It uses the <code>PlayerAdded</code> event to trigger the <code>welcomePlayer</code> function, demonstrating how events and functions work in Lua.</p>

<h3>
  
  
  Understanding Variables and Conditions
</h3>

<p>Variables and conditional statements are fundamental in controlling logic:<br>
</p>

<div class="highlight js-code-highlight">
<pre class="highlight lua"><code><span class="kd">local</span> <span class="n">health</span> <span class="o">=</span> <span class="mi">100</span>

<span class="k">if</span> <span class="n">health</span> <span class="o">&gt;</span> <span class="mi">0</span> <span class="k">then</span>
    <span class="nb">print</span><span class="p">(</span><span class="s2">"Player is alive"</span><span class="p">)</span>
<span class="k">else</span>
    <span class="nb">print</span><span class="p">(</span><span class="s2">"Player is dead"</span><span class="p">)</span>
<span class="k">end</span>
</code></pre>

</div>



<p>This code snippet checks a player’s health and prints a message based on the condition. Understanding conditional logic is crucial for making decisions in your game scripts.</p>
