<p>Claude Code shows a spinner while thinking. By default it says things like "Thinking...", "Processing...". boring. <br>
So I made a small CLI tool to replace them with themed quotes.</p>

<p><a href="https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Ftjemv6vs0f21gxqbrt9z.gif" class="article-body-image-wrapper"><img src="https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Ftjemv6vs0f21gxqbrt9z.gif" alt="claude-verbs-demo-gif" width="750" height="155"></a></p>

<h2>
  
  
  Usage
</h2>

<p><code>npx claude-verbs</code><br>
Pick a theme, done. Works with both Claude Code CLI and VSCode extension.</p>

<h2>
  
  
  Themes
</h2>

<ul>
<li>Star Wars: "May the tokens be with you"</li>
<li>LOTR: "Consulting the elves"</li>
<li>Dune: "Tokens must flow"</li>
<li>Matrix: "Downloading kung fu"</li>
<li>Breaking Bad: "99.1% pure coverage"</li>
<li>Game of Thrones: "What do we say to PO? Not today."</li>
</ul>

<h2>
  
  
  Why
</h2>

<p>Because if you're going to stare at a spinner all day, it might as well make you smile :)</p>
