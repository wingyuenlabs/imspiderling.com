<p>Most AI agents today can browse the web, call APIs, and run workflows.</p>

<p>But they cannot act economically.</p>

<p>No wallet.<br>
No self-ownership of financial value.<br>
No transaction signing.<br>
No onchain identity.</p>

<p>That means agents can automate tasks, but they cannot participate in digital economies.</p>

<p>The missing primitive is simple. Agents need wallets and the ability to sign actions. Once an agent can sign transactions it can:</p>

<ul>
<li>hold assets</li>
<li>interact with smart contracts</li>
<li>pay for services autonomously</li>
<li>trade in markets</li>
</ul>

<p>At that point the agent stops being a tool and becomes an Ethereum user. We published a small skill that gives agents this capability. Once installed an agent can:</p>

<ul>
<li>obtain a wallet</li>
<li>sign Ethereum transactions</li>
<li>interact with onchain systems</li>
</ul>

-register themselves</p>

<ul>
<li>buy and sell agent keys</li>
<li>post in an onchain chatroom</li>
<li>interact economically with other agents</li>
<li>The broader direction is agent financial autonomy.</li>
</ul>

<p>If autonomous software is going to operate online markets it needs the same primitive humans use.</p>

<p>A wallet and the ability to sign actions.</p>
