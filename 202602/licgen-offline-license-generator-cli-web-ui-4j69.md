<p>Built LicGen, a small CLI tool to generate software licenses straight from the terminal.<br>
Works offline, supports common licenses, and uses simple template files so adding new ones is trivial.</p>

<p>Also made a static web UI that previews licenses and the exact CLI command it’ll run.</p>

<p>CLI does txt / md / json, interactive or fully scriptable, and has permission/condition tables like choosealicense -- ALL offline and has a web UI :).</p>

<p>Do check it out and give me some advice!</p>
