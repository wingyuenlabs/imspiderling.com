<h1>
  
  
  Smart Multimodal Invoice &amp; Expense Analyzer
</h1>

<h2>
  
  
  Description
</h2>

<p>This web app leverages Google Gemini 2.5 multimodal capabilities to analyze invoices, receipts, and expense records from images, videos, or audio recordings. It extracts structured data, categorizes expenses, detects anomalies, and generates interactive reports for users.</p>

<h2>
  
  
  Features
</h2>

<ul>
<li>Upload image, video, or audio of receipts/invoices</li>
<li>Automatic extraction: vendor, date, items, total, taxes, currency</li>
<li>Categorizes expenses: personal, business, tax-related</li>
<li>Detects anomalies: duplicates, missing data, incorrect totals</li>
<li>Generates interactive summaries and reports</li>
</ul>

<h2>
  
  
  Technology Stack
</h2>

<ul>
<li>Google AI Studio (Gemini 2.5 Flash/Pro)</li>
<li>Cloud Run for deployment</li>
<li>Frontend: HTML + JavaScript</li>
<li>Backend: Python Flask</li>
</ul>

<h2>
  
  
  How to Use
</h2>

<ol>
<li>Upload your invoice, receipt, or expense video/audio
</li>
<li>AI parses and displays structured data
</li>
<li>View categorized summary and anomaly detection
</li>
<li>Download or share the report</li>
</ol>
