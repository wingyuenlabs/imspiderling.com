<p>Un <a href="https://developer.apple.com/documentation/swiftui/securefield" rel="noopener noreferrer"><code>SecureField</code></a> permite al usuario insertar una línea de texto, sin embargo, esta vista reemplaza los caracteres insertados por el usuario por puntos para ocultar información sensible como contraseñas.</p>

<ul>
<li>
<a href="https://developer.apple.com/documentation/swiftui/securefield/init(_:text:)" rel="noopener noreferrer"><code>init(_:text:)</code></a>: Recibe un placeholder y un binding hacia una variable que almacena el texto.</li>
</ul>
