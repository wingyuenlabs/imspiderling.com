<p>Computing has advanced over time and is now essential across industries to help organizations make better decisions and boost profits through data analysis. A simple tool like Microsoft Excel serves as an excellent starting point for analyzing various datasets, including medical records, sales data, student performance, and social media metrics. </p>

<h2>
  
  
  Data Analysis Basics
</h2>

<p>Data analysis involves examining, cleaning, organizing, and interpreting data to uncover useful insights, identify patterns, and support decision-making.</p>

<h3>
  
  
  Why Clean Your Data?
</h3>

<ul>
<li>Ensures high integrity by removing biases.</li>
<li>Enables correct use of formulas.</li>
<li>Aligns with the problem statement.</li>
<li>Maximizes use of complete sample size.</li>
</ul>

<h2>
  
  
  Types of Data Analysis
</h2>

<ol>
<li>
<strong>Prescriptive</strong> - Uses historical data to recommend actions.</li>
<li>
<strong>Predictive</strong> - Analyzes past patterns to forecast the future.</li>
<li>
<strong>Diagnostic</strong> - Explains <em>why</em> something is happening.</li>
</ol>

<h2>
  
  
  Logical Functions
</h2>

<p>Logical functions test conditions and return results based on TRUE/FALSE.</p>

<h3>
  
  
  1. IF &amp; IFS
</h3>

<ul>
<li>
<strong>IF</strong>: <code>=IF(logical_test, value_if_true, value_if_false)</code>
</li>
<li>
<strong>IFS</strong>: Multiple conditions <code>=IFS(condition1, result1, condition2, result2, ...)</code>
</li>
</ul>

<p>Examples:</p>

<p>=IF(G2&gt;30,"OLD","YOUNG")<br>
=IFS(G2&gt;50,"OLD", G2&gt;40,"ABOVE-MEDIUM", G2&gt;30,"MEDIUM","YOUNG")<br>
<a href="https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fjqikz1f1b2t73xd2fbu3.png" class="article-body-image-wrapper"><img src="https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fjqikz1f1b2t73xd2fbu3.png" alt=" " width="800" height="247"></a></p>

<h3>
  
  
  2. AND &amp; OR
</h3>

<ul>
<li>
<strong>AND</strong>: All conditions must be true.</li>
<li>
<strong>OR</strong>: Any condition can be true.</li>
</ul>

<p>Examples:</p>

<p>=IF(AND(G2&gt;40,E2&gt;70000),"H--R","OTHERS")<br>
=IF(OR(G2&gt;40,E2&gt;70000),"H--R","OTHERS")</p>

<p><a href="https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2F7oputfo55qx91zfz7ydu.png" class="article-body-image-wrapper"><img src="https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2F7oputfo55qx91zfz7ydu.png" alt=" " width="800" height="165"></a></p>

<h3>
  
  
  3. NOT
</h3>

<p>Negates a condition: <code>=NOT(logical_test)</code> or use <code>&lt;&gt;</code>.</p>

<p>Other operators: <code>&lt;</code>, <code>&gt;</code>, <code>=</code>, <code>&lt;&gt;</code>, <code>&lt;=</code>, <code>&gt;=</code>.</p>

<h2>
  
  
  Data Filtering &amp; Sorting
</h2>

<ul>
<li>
<strong>Sorting</strong>: Arranges data (A-Z, newest-oldest, etc.).</li>
<li>
<strong>Filtering</strong>: Shows data meeting specific conditions.</li>
</ul>

<p><strong>Filter Function Example</strong>:</p>

<p>=FILTER(A2:F10, D2:D10="HR")</p>
