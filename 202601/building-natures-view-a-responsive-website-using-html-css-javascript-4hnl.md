<p>I recently built a project called <strong>Nature’s View</strong>, a responsive website focused on raising awareness about nature while practicing clean frontend development.</p>

<p>The main goal was to build something that looks good on <strong>desktop and mobile</strong>, without using frameworks — just <strong>HTML, CSS Grid, and vanilla JavaScript</strong>.</p>

<p><strong>Tech Stack</strong></p>

<ul>
<li>HTML5 (semantic layout)</li>
<li>CSS Grid &amp; Flexbox</li>
<li>Vanilla JavaScript</li>
<li>Mobile-responsive design</li>
</ul>

<p><strong>What I Focused On</strong></p>

<ul>
<li>Responsive layout (desktop → mobile)</li>
<li>Image slideshow without libraries</li>
<li>Theme toggling (light / dark)</li>
<li>Clean, readable CSS</li>
</ul>

<p><strong>Challenges I Faced</strong><br>
One challenge was making the layout work on smaller screens.<br>
The desktop grid looked fine, but on mobile the sidebar and slideshow needed rethinking.</p>

<p>This helped me understand <strong>media queries</strong>, <strong>flexible layouts</strong>, and why designing mobile-first matters.</p>

<p><strong>Why This Project Matters</strong><br>
This project isn’t just about code — it’s about using web development to support <strong>environmental awareness</strong>.</p>
